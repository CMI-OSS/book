CMI만의 개발 레퍼런스 모음집 만들기~ 참고

  • "[Prefix] 제목" 형태로 작성
  • 개발에 도움이 될만한 레퍼런스들을 카테고리별로 정리
  • 일단 무엇이든 적고! 나중에 한번씩 정리할계획
  • 궁금한거 있으면 무엇이든 언제든지 물어보기

Prefix

  • BLG : 블로그 포스팅(ex. 네이버 블로그, 티스토리, 벨로그 등)
  • YTB : 유튜브 영상
  • BOOK : 책
  • LEC : 동영상 강의(인프런, 유데미 등)
  • WEB : 그 외 웹사이트
